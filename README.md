# Automation Test Project

## Project Overview

This project contains automated test scripts developed using Selenium WebDriver to validate key functionalities of a web application.

## Tools and Technologies Used

* Python
* Selenium WebDriver
* PyTest 
* Chrome Browser
* Git 

## Prerequisites

* Python 3.x installed
* pip installed
* Google Chrome browser

## Setup Steps

1. Clone the repository:
   git clone https://github.com/your-username/automation-test.git

2. Navigate to project folder:
   cd automation-test

## Test Execution Steps

Run tests using:

pytest

## Limitations

* Tests run only on Chrome browser
* Internet connection required
